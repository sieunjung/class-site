Tide Ties

I was researching to build a translation machine that allows anyone to create different visuals and outcomes, avoiding a defined result. A pegboard is a machine that translates different patterns and shapes using pegs and strings according to rules that I created. This book consists of the instructions and outcomes of using the pegboard as a translation machine. The translation machine is not only for an artist or designer but can be a fun activity for an individual and team.
